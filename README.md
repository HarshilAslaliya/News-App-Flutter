## 🗒 About

This is News app where you can read category wise and daily News

## ⚙️ Features

- Main features are as follows:

    - Dark & Light Theme
    - Categorywise News
    - Daily News
    - Call API using dio package
    - Use provider state management
    - MVC directory structure
